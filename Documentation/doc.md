
ОБЩИЙ ВИД ПРИЛОЖЕНИЯ
Основной рабочий экран это банер-название вверху.
Далее 2 большие области, левая занимает 2/3 панели, с днями недели - календарь. 
Правая остальное место и везде отступы в круг. Тут небольшие квадраты с блюдами на выбор.


**Сущности:**

ПОЛЬЗОВАТЕЛЬ (User)
Имя 
пароль
номер телефона для восстановления пароля
фото
*Привязка к облаку соц.сетям

БЛЮДО (Dish)
Поле Название блюда.
Сохранение картинки большой и. маленькой для переноски в день.
*Ссылка на рецепт.
*Список покупок. + на сколько разных блюд нужен продукт (пример: "Используется в 2 рецептах").
Блюдо можно создать, редактировать, удалить, *добавить картинку.

Отображение Блюда:
Квадрат с картинкой сверху и названием внизу. У названия есть ограничение на 20(?) символов далее вставляется ...


ДЕНЬ НЕДЕЛИ (Day)
Меню на день, можно сюда перетащить и блюдо сохранится следующим в списке.
Название дня недели.
*день календаря (Date)
Может сохранять в себе несколько блюд. (ArrayList объектов<Блюдо>).
*Массив их названий и ID - линк для отображения общего списка и перехода на рецепт.

Отображение дня недели:
Прямоугольник, сверху название дня недели (* дата) далее пусто и бледная надпить по дефолту: "Перенесите блюдо"/ "Выберите блюдо" / "Добавьте блюдо"

Все поля отмеченный символом * на данный момент не являются приоритетом и могут быть реализованы позже.
